## 💻 Terminal Commands (For Quick Practice)

All commands from the exercises in one place:

### Basic Workflow
```bash
git status              # See what changed
git diff filename       # See specific changes
git add .              # Stage all changes
git commit -m "msg"    # Create snapshot
git log --oneline -5   # View history
```

### Branches
```bash
git branch                          # List branches
git branch newfeature               # Create branch
git checkout newfeature             # Switch branch
git checkout -b feature/name        # Create & switch
git merge feature/name              # Merge into current
git branch -d oldfeature            # Delete branch
```

### Undoing
```bash
git restore filename                # Undo unstaged changes
git restore --staged filename       # Unstage file
git reset --soft HEAD~1             # Undo last commit
git revert HEAD                     # Safely undo published commit
git reflog                          # Find lost commits
```

### GitHub
```bash
git push origin main                # Push to GitHub
git pull origin main                # Pull from GitHub
git remote -v                       # See remotes
git tag v1.0.0                      # Create tag
```
